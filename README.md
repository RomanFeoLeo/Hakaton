# Hakaton

Веб-приложение для мониторинга и замены уличных ламп с помощью дрона.

## Что внутри

Проект находится в папке `web/` и включает 2 режима интерфейса:

- `3D анимация`: сцена на Three.js с лампами, дроном, погодой (дождь/снег), дорожкой и декором.
- `Карта`: интерактивная карта (Leaflet) с выбором сломанной лампы, планированием и запуском замены.

## Основные возможности

- WebSocket-обмен между клиентом и сервером (`ws://localhost:3000`).
- Управление статусами ламп (`working`, `broken`, `scheduled`, `in_progress`).
- Имитация полета дрона к выбранной лампе и возврата на базу.
- Панель задач с текущим состоянием замен.
- В 3D-сцене выбор лампы кликом по фонарю с автоматической подстановкой в пульт.

## Структура

- `web/server.js` — Node.js сервер (Express + WebSocket).
- `web/public/index.html` — интерфейс с вкладками `3D анимация` и `Карта`.
- `web/public/js/main.js` — 3D-сцена и логика управления в Three.js.
- `web/public/js/map-tab.js` — логика вкладки карты (Leaflet).
- `web/models/` — 3D-ресурсы модели.

## Быстрый запуск

Требования:

- Node.js 18+ (рекомендуется LTS)

Команды:

```bash
cd web
npm install
npm start
```

После запуска откройте:

- `http://localhost:3000`

## Режим разработки

```bash
cd web
npm run dev
```
